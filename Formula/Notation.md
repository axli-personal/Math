## Basic Notation

| Format                   | Display                  |
| ------------------------ | ------------------------ |
| `\eta`                   | $\eta$                   |
| `\alpha`                 | $\alpha$                 |
| `\beta`                  | $\beta$                  |
| `\epsilon`               | $\epsilon$               |
| `\theta`                 | $\theta$                 |
| `\infty`                 | $\infty$                 |
| `\sum_a^b`               | $\sum_a^b$               |
| `\lim_{x \to 0}`         | $\lim_{x \to 0}$         |
| `\int_a^b f(x) \, dx`    | $\int_a^b f(x) \, dx$    |
| `a \cdot b`              | $a \cdot b$              |
| `a \times b`             | $a \times b$             |
| `a \div b`               | $a \div b$               |
| `\frac{a}{b}`            | $\frac{a}{b}$            |
| `\text{It looks like: }` | $\text{It looks like: }$ |
